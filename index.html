Fireable

Untitled C++ based rhythm game

A project made for Tredu in 2022-2023.
Made using SDL2 and SDL_GPU, made within a few months without any prior experience in C++.
The game is a rhythm game, where the player has to hit the correct notes at the correct time.

I did not have a lot of time to implement all the features I wanted, but I am still proud of the result.
Game is also surprisingly well optimized, although for some reason struggles on NVIDIA GPUs.

Source code: Github
